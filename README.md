🎁 Amigo Secreto
Um aplicativo web simples para sortear amigos secretos entre um grupo de pessoas.

📋 Descrição
Este projeto permite que você:

Adicione nomes de amigos
Veja a lista de participantes
Sorteie um amigo secreto aleatório

🚀 Funcionalidades
Adicionar Amigos: Digite nomes e clique em "Adicionar" para incluir na lista
Validação: Impede nomes vazios e repetidos
Lista Visual: Mostra todos os participantes em tempo real
Sorteio Aleatório: Seleciona um amigo secreto de forma aleatória

🛠️ Tecnologias Utilizadas
HTML5: Estrutura da página
CSS3: Estilização e layout
JavaScript: Lógica e interatividade

🎯 Como Usar
Adicionar Participantes:
Digite o nome de um amigo no campo de texto
Clique no botão "Adicionar"
Repita até incluir todos os participantes
Realizar o Sorteio:
Clique no botão "Sortear amigo"
Um nome será selecionado aleatoriamente
